Agricultural Autonomous Driving Dataset (Preview)

Smart Farming Lab
Leipzig University

Abstract

We present a large (100GB+) comprehensive multi-modal dataset for autonomous driving in agricultural environments, captured using a sophisticated sensor setup including multiple cameras, LiDAR, GPS, and IMU sensors mounted on a Fendt 728 Variotractor.

The dataset includes synchronized data from: 6 cameras (rear left/mid/right, side left/right, stereo ZED), LiDAR point clouds from an Ouster sensor, high-precision GPS positioning with Novatel OEM7, and IMU measurements for comprehensive environmental perception.

This dataset is designed to support research in agricultural robotics, autonomous navigation in unstructured environments, computer vision, environmental mapping and multi-modal sensor fusion.

Dataset Overview

Sensors

  • 6 RGB Cameras (multiple viewpoints)
  • Ouster LiDAR (3D point clouds)
  • Novatel OEM7 GPS/INS
  • High-frequency IMU

Data Characteristics

  • Location: Lehr- und Forschungsgut Oberholz der Universität Leipzig
  • Seasons: Currently Spring and Summer, will be expanded to include Fall and Winter
  • Tasks: Plowing, tilling, will be expanded to include planting, harvesting, and more

Precision LIDAR Point Clouds

The Ouster OS1 LiDAR (128 scanlines) is mounted 60° downward-facing at 2m height. This enables a high-resolution 3D point cloud of the environment that can be used for precise mapping and localization, for object detection and classification and as a deterministic safety sensor.
